Projects

The MODES group has been involved in a number of European and industrial projects. Below is a data-driven list of representative projects.

  • OSSMETER (2021)

    Open-source software quality assessment and metrics tooling. We developed automated analyses and dashboards to help projects track metrics over time.

    Partners: University of L'Aquila, Rete Ferroviaria Italiana

  • TYPHON (2020)

    A research project on automated program analysis and model mining focused on scalable, language-agnostic approaches.

    Partners: European Consortium

  • CROSSMINER (2019)

    Cross-repository mining and analysis tools for extracting usage patterns and API evolution trends.

    Partners: Industrial Partner A
